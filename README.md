# TranIU-Net: Indicative Electrical Tomography Imaging Based on Implicit Unrolling Transformer
This repository provides a PyTorch implementation of TranIU-Net for electrical tomography image reconstruction. The reproduction code was rewritten for this repository. The implementation was refactored for readability and simplicity, with clearer structure and naming.
